---
Note :</br>
[sudo] password for kali:kali

---
> sudo apt install adduser -y

* Add Username
```
sudo adduser (username)
```

> Create a new password.

> Press enter to skip adding data.

> If there is a (y/n) option, select y.

* Login user
```
su (username)
```

* Logout user
```
exit
```

* Delete Username
```
sudo deluser (username)
```
